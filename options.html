Fast Copy

Copy images from any page to the clipboard.

Must differ from the image modifier above.
Lets you paste into rich editors (Gmail, Docs) keeping the image.
Requires the Local REST API plugin. On copy, the image is saved to your vault and an ![[…]] embed is appended to the active note.
Enable “Non-encrypted (HTTP) Server” in the plugin and use the HTTP option — the browser rejects the plugin’s self-signed HTTPS cert.
Copy it from the Local REST API plugin settings.
Off = only save the file, don’t touch any note.
