# Fast Copy

Расширение для браузеров на основе Chromium (Chrome, Edge, Brave, Opera) для
копирования изображений с веб-страницы в буфер обмена.

> 🇬🇧 English version: [README.md](README.md)

## Как пользоваться

1. Зажмите **Ctrl** и **кликните левой кнопкой** по любой картинке на странице.
2. Появится небольшое уведомление «Image copied» — изображение скопировано в
   буфер обмена в формате PNG.
3. Вставьте его (`Ctrl+V`) в любой редактор, чат, документ и т. д.

Работает с:
- элементами `<img>` (в том числе картинками внутри ссылок — переход по ссылке
  подавляется);
- фоновыми изображениями CSS (`background-image`);
- `<canvas>` (если он не «испорчен» кросс-доменным содержимым).

## Установка (распакованное расширение)

1. Откройте `chrome://extensions` (или `edge://extensions`, `brave://extensions`).
2. Включите **Режим разработчика** (переключатель в правом верхнем углу).
3. Нажмите **Загрузить распакованное расширение** и выберите папку проекта
   (`fast_copy`).
4. Готово — откройте любую страницу и кликните по картинке с зажатым Ctrl.

## Настройки

Кликните по иконке расширения на панели инструментов (или откройте через
`chrome://extensions` → *Подробнее* → *Параметры расширения*), чтобы настроить:

- **Клавиша копирования картинки** — какая клавиша + левый клик копируют
  изображение (Ctrl / Alt / Shift).
- **Клавиша копирования URL картинки** — другая клавиша + левый клик копируют
  адрес картинки как обычный текст (или *Отключено*). Должна отличаться от
  клавиши копирования картинки.
- **Дублировать как HTML `<img>`** — дополнительно помещает в буфер тег `<img>`,
  чтобы можно было вставлять картинку в редакторы с форматированием (Gmail,
  Google Документы и т. п.).
- **Показывать уведомление** — включение/выключение всплывающего уведомления на
  странице.

Настройки хранятся в `chrome.storage.sync` и применяются мгновенно, без
перезагрузки страницы.

## Как это работает

- `content.js` отслеживает клик с модификатором по изображению и записывает
  результат в буфер обмена через асинхронный Clipboard API.
- `background.js` (service worker) скачивает изображение — обходя CORS-ограничения
  страницы за счёт `host_permissions` — и конвертирует его в PNG, формат, который
  буфер обмена принимает надёжно.

## Разрешения

- `clipboardWrite` — чтобы помещать изображение в буфер обмена.
- `storage` — чтобы сохранять ваши настройки.
- `host_permissions: <all_urls>` — чтобы service worker мог скачивать картинки с
  любого домена.

## Ограничения

- На служебных страницах (`chrome://`, интернет-магазин расширений, просмотрщик
  PDF) контент-скрипты не работают — это ограничение браузера.
- `<canvas>`, «испорченный» кросс-доменным содержимым без CORS, скопировать
  нельзя — это запрет безопасности браузера.

## Файлы

| Файл            | Назначение                                              |
| --------------- | ------------------------------------------------------- |
| `manifest.json` | Конфигурация Manifest V3                                |
| `content.js`    | Обработчик клика, запись в буфер, всплывающее уведомление |
| `background.js` | Скачивание картинки + конвертация в PNG, открытие настроек |
| `options.html`  | Разметка страницы настроек                              |
| `options.js`    | Логика страницы настроек (`chrome.storage.sync`)        |
| `icons/`        | Иконки расширения (16/48/128 px)                        |
